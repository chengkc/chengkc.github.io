书本推荐

《作为意志和表象的世界》全书共分四篇，第一篇可被看作是全书的导论，也可以和第二篇合起来看作全书的第一部分，主要讨论认识论和“真”的问题。第三篇关涉“美”的问题，而第四篇是关于“善”和人生的终极关怀问题的。

《作为意志和表象的世界》是叔本华思想发展的顶点。叔本华思想主要受柏拉图、康德和印度佛学的影响。书中叔本华认为“理性”不过是意志的派生物，意志具有决定性。

人性一个最特别的弱点是：在意别人如何看待自己。
